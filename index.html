About

Created by Rob Sim because he needed to create icons for iOS 7 in a simple way. The main template, the "Slicy + .PSD" one requires the third party app Slicy which you should already have, but if you don't go scoop it up. It's totally worth it, and makes using this template to create all of your icons a breeze. You just edit the Smart object once at it will change in every location! If you prefer Sketch or just plain old photoshop, there are templates for those too!

These are free to use, but please do not distribute them! Enjoy!
